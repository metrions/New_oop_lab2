Справка по использованию телефонной книги

1. Добавление контакта

Для добавления нового контакта:

  • Введите ФИО и номер телефона в соответствующие поля.
  • Нажмите кнопку "Добавить".

2. Удаление контакта

Чтобы удалить контакт:

  • Выберите контакт, кликнув по нему в таблице.
  • Нажмите кнопку "Удалить".

3. Редактирование контакта

Чтобы изменить контакт:

  • Выберите контакт в таблице.
  • Нажмите кнопку "Изменить".
  • В открывшемся окне измените ФИО или номер телефона.
  • Нажмите "Сохранить".

4. Очистка списка

Чтобы удалить все контакты сразу, нажмите кнопку "Очистить".

5. Поиск контакта

Для поиска контакта:

  • Введите ФИО или номер телефона в форму поиска.
  • Нажмите кнопку "Найти".

6. Импорт контактов

Чтобы импортировать контакты из CSV-файла:

  • Нажмите кнопку "Импортировать".
  • Выберите CSV-файл на компьютере.

7. Экспорт контактов

Чтобы сохранить контакты в CSV-файл:

  • Нажмите кнопку "Выбрать файл и сохранить".
  • Выберите папку и имя файла или создайте новый.
