Как использовать конвертер

Этот конвертер позволяет вам преобразовывать числа из одной системы счисления в другую. Поддерживаемые системы счисления: от 2 до 16.

Шаги для использования:

  1. Введите число в поле "Исходное число".
  2. Выберите систему счисления для исходного числа в поле "Основание с. сч.". Это может быть от 2 до 16.
  3. Выберите систему счисления для результата в поле "Основание с. сч. результата".
  4. Нажмите кнопку "Конвертировать", чтобы преобразовать число.

Справка по кнопкам:

  • Цифры (0-9, A-F): используйте эти кнопки для ввода цифр в числе.
  • Разделитель: кнопка с точкой (.) добавляет десятичный разделитель.
  • BS (Backspace): удаляет последний введенный символ.
  • CE (Очистить): очищает все введенные данные и сбрасывает поля.

История Операций:

Каждое преобразование сохраняется в истории. Вы можете очистить историю, нажав на кнопку "Очистить историю" на странице истории.
